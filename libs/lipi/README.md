# Lipi

Lipi is a [specification for a data serialization format](specification.md), in which data is represented as a compact binary object.

The word ["Lipi"](https://en.wikipedia.org/wiki/Lipi_(script)) (লিপি; लिपि) means **script**, **writing**, **alphabet**.  
It is a writing system or set of symbols, through which ideas, language, or something are represented in written form.

## Implementations

This repository contains multiple language implementations of the [Lipi serialization format](specification.md).
